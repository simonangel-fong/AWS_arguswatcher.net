Django ECDeployer

Django ECDeployer is a streamlined solution designed to simplify the deployment process of Django applications on AWS EC2 instances. Ideal for developers looking to deploy Django applications on AWS without the complexities of manual setup.

Project GitHub

Features

Simplified Deployment

To simplify the Django project deployment, users are required to provide only two key arguments - the name of the Django project and the GitHub repository URL.

AWS Cloud

Utilizes a predefined EC2 launch template with user data to host the Django application.
Leverages AWS SDK boto3 to provision infractures.

CI/CD

This project uses AWS CodeBuild, CodePipeline, and CodeDeploy Integration. Automates the entire deployment pipeline, from source code changes to EC2 instance provisioning and application deployment.

Demo Video

Usage

Fill the form to create a server
List all running servers in a table
View server information in detail page
Visit your Django project via public IP
Click "Terminate" button to terminate server
Confirm termination
Terminated server is remove from the list
